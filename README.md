## 📌 Overview

Recruiters often spend significant time manually reviewing resumes. This project automates the screening process by leveraging NLP and Machine Learning to classify resumes into the most relevant job categories.

The system processes raw resume text, extracts meaningful features using TF-IDF, and predicts the appropriate job role category with high accuracy.

---

## ✨ Key Features

✅ Resume Text Cleaning & Preprocessing

✅ NLP-based Feature Extraction (TF-IDF)

✅ Handling Class Imbalance using Oversampling

✅ Label Encoding for Multi-Class Classification

✅ Machine Learning-based Resume Classification

✅ Real-Time Resume Category Prediction

✅ Model Serialization using Pickle
